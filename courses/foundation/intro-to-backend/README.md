## Learning goals

- [ ] Introduction to Node.js

- [ ] Building a simple webserver

- [ ] Sending requests

- [ ] Database interaction
  - [ ] What is CRUD (Create, Read, Update, Delete)
  - [ ] CRUD vs REST
  - [ ] Connecting to a database using JavaScript
  - [ ] Executing queries (i.e. CRUD)
